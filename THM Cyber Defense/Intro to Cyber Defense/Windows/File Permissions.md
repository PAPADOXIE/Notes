## File Permissions

### Can be applied to
- Users
- Groups

### Permissions
- Full Control (F)
- Modify (M)
- Read and Execute (RX)
- List Folders Content
- Read
- Write (WD)
- Special Permissions

##### Check Permissions
	icacls <[File][Directory]>

##### Set Permissions
	icacls <[File][Directory]> /setowner <User>
